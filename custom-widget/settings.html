How to Update Configuration

To change widget settings, update these environment variables in your .env file and restart the server:

  • WIDGET_NAME - Display name for the widget
  • WIDGET_PRIMARY_COLOR - Hex color for buttons and highlights (e.g., #2c5530)
  • WIDGET_ALLOWED_DOMAINS - Comma-separated domains or * for all domains
